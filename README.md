#### Basic Commands to Start & Activate my Project...

cd project

pipenv install venv

\venv\Scripts>activate/deactivate

python -m venv venv or virtualenv <"nombre del projecto">

venv\Scripts>activate

\venv\Scripts>pipenv shell  or  (>exit) to exi
